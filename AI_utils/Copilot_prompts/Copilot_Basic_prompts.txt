##10 Copy-Paste Copilot Prompts

These are my go-to Microsoft Copilot prompts that work across different situations. Just copy, paste, and adjust the specifics:

1. “Rewrite this to be 40% shorter while keeping all the important information” — Perfect for cutting fluff from drafts
2. “Turn this into an executive summary with exactly 3 key takeaways” — Great for busy stakeholders who won’t read full reports

3. “Create a comparison table showing pros and cons of these three options” — Makes decisionmaking clearer

4. “Explain this [Excel formula/process/concept] step-by-step for someone who’s never used it before” — Builds understanding instead of just providing answers

5. “Draft a friendly but professional email declining [specific request] and offering [specific alternative]” — Maintains relationships while setting boundaries

6. “Convert these messy meeting notes into a clean action item list with owners and deadlines” — Transforms rambling notes into useful project management

7. “Summarize this report in language that someone outside our industry would understand” — Bridges communication gaps with clients or stakeholders

8. “Generate 5 different subject lines for this email, ranging from direct to creative” — Gives you options for different contexts

9. “Create a 30-minute meeting agenda focused on solving [specific problem]” — Keeps discussions productive and goal-oriented

10. “Identify the weakest arguments in this proposal and suggest how to strengthen them” — Helps you stress-test ideas before presenting

##Productivity & M365 Integration

1. Executive Email Drafting
"Draft a reply to [Sender Name]'s email about the [Project Name] delay. Maintain a professional, empathetic, and solution-oriented tone. Acknowledge their concerns, outline our next steps to get back on track, and suggest a 15-minute sync for [Day] morning."

2. Meeting Synthesis & Action Items
"Analyze the transcript from the last [Project Name] meeting in Teams. Create a clean summary focusing on key decisions, unblocking issues, and clearly assigned action items with the names of the owners."

3. Data Analysis in Excel
"Analyze this sales data table. What were the top 3 performing product categories in the last quarter? Identify any anomalous drops in sales, format the findings as a narrative executive summary, and create a chart recommendation."

4. Document Transformation in Word
"Take this /[Document Name] and transform it into a professional, engaging onboarding guide for new team members. Change the formatting to include bullet points, bold key terms, and end with a quick knowledge check quiz."

5. Presentation Outline from a Doc
"Using the attached /[Proposal Name], create an 8-slide presentation outline for an executive board meeting. For each slide, provide the main title, 3 bullet points of content, and speaker notes that explain the strategic value."

##Creativity & Content Creation
Unblock writer's block and generate high-quality marketing, writing, or brainstorming materials.

6. Marketing Campaign Brainstorm
"Act as a Senior Growth Marketer. Brainstorm 5 out-of-the-box, low-budget marketing angles for our new [Product/Service] targeting [Target Audience]. For each idea, include the core concept, the primary channel, and a sample ad headline."

7. Blog Post Drafting & SEO Optimization
"Write a 1,000-word blog post about [Topic] targeting [Industry]. Include an H2 and H3 structure, engaging introduction, and an SEO meta description. Maintain an authoritative but highly accessible tone."

8. Social Media Repurposing
"Take this/[Document Name] and convert it into a 5-day social media drip campaign. Create distinct posts for LinkedIn, X (Twitter), and Instagram. Ensure the LinkedIn posts are professional, X is conversational/punchy, and Instagram relies on visual/caption formatting."

9. Persona-Based Writing
"Rewrite this product announcement so it appeals to both non-technical end users and IT professionals. Clearly separate the value propositions for both personas into two distinct sections."

10. Creative Storytelling & Scripting
"Write a 60-second script for a promotional video explaining [Product]. Focus on the pain points our customers face and the emotional relief our product provides. Include visual cues for the camera in brackets."

##Coding & Technical Tasks
Use these prompts in GitHub Copilot or Copilot Chat to debug, write, and optimize code quickly.

11. Code Explanation & Documentation
"Analyze this code block: [Paste Code]. Explain exactly what this code does, line by line. Then, rewrite it to include proper commenting, docstrings, and a summary of its time and space complexity."

12. Refactoring for Readability & Performance
"Refactor this [Language] function [Paste Code] to make it more [DRY/performant/idiomatic]. Remove any redundant steps and provide a Before/After comparison with a brief explanation of why the new code is better."

13. Test Driven Development (TDD) Generation
"Based on this [Language] class/module [Paste Code], write a comprehensive suite of unit tests using the [pytest/Jest/JUnit] framework. Include tests for edge cases and invalid inputs."

14. SQL Query Generation
"Write an optimized SQL query that pulls [Specific Data Requirement] from a database with tables [Table1] and [Table2]. Join them on [Key] and filter the results where [Condition] is true."

15. Debugging & Error Translation
"I am getting the following error in my application: [Paste Error]. The function attempting to run is [Paste Code]. What is the root cause of this error, and what are 2 possible ways to fix it? Provide the updated code for the best solution."

16. Regex & String Parsing
"Write a [Language] regular expression that matches a [Specific Pattern, e.g., standard email format with specific TLDs]. Provide a brief explanation of how the regex operates and 3 test cases that pass and 3 that fail."


##Pro-Tip for Better Copilot Results
To ensure Copilot gives you the most precise response, use the C.I.C.O. framework:

- Context: Tell the AI who it is acting as (e.g., Act as a Senior Python Developer).
- Intent: Be very clear about your exact goal (e.g., Write a unit test).
- Constraints: Add limits like word count, tone, or frameworks to use.
- Output Format: Specify how you want the data delivered (e.g., Output in Markdown table format).

##Copilot Chat work together as a specialized AI pair programming

1. Debugging Code
Copilot Chat is the primary tool for diagnosing and fixing errors. It leverages contextual awareness of your code, variables, and call stacks. 

YouTube
·Microsoft Visual Studio

Explain Errors: Highlight an error in your terminal or editor and use the /explain command to get a breakdown of why it's happening.
Propose Fixes: Use the /fix command to automatically generate a suggested correction for a specific block of code.
Debugger Integration: In Visual Studio, you can use the Analyze with Copilot button when the application is paused on an exception to get an immediate assessment and a suggested fix.
Variable Context: You can right-click variables in the "Autos" or "Locals" window and select Ask Copilot to investigate specific runtime values. 

2. Writing New Code
Both tools assist in code creation but serve different levels of granularity. 
Inline Suggestions: As you write, GitHub Copilot suggests the next line or entire function blocks based on comments and existing patterns in your file.
Natural Language to Code: Use Copilot Chat to describe a complex feature in plain English, and it will generate the necessary boilerplate and logic for you to review and insert.
Boilerplate & Patterns: Copilot excels at finishing repetitive loops, data structures, and simple functions quickly to keep you in "flow".

3. Optimizing & Refactoring
Copilot helps clean up and enhance existing code without changing its core behavior. 

GitHub Docs
Performance Improvements: Ask Copilot Chat to "Improve the performance of this code" or "Make this code more concise".
Slash Commands: Use /optimize (in some versions) or prompts like "Refactor this code for better readability" to see a side-by-side comparison of the changes before applying them.
Unit Test Generation: Use the /tests command to create test cases for your functions, ensuring that optimizations don't break existing functionality. 

Quick Tips for Better Results

-Use Specific Context: In Copilot Chat, use the # symbol to reference specific files or code selections (e.g., "Explain how #selection works").
-Iterative Prompting: If the first suggestion isn't perfect, treat Copilot Chat like a tutor and ask follow-up questions to refine the result.
-Verify Outputs: Always review and test AI-generated code; while powerful, Copilot can sometimes suggest suboptimal or incorrect logic that requires developer oversight.
